About this application
This demonstration application is a companion resource to the "ArcGIS Maps SDK for JavaScript: Bring in Data from Anywhere" session at Esri Developer & Technology Summit. This example illustrates.
  • Using the request module.
  • Creating GeoJSON layers.
Learn more about HTTP requests and GeoJSONLayers in the ArcGIS Maps SDK for JavaScript in the documentation.
